# محول PDF الذكي

موقع ويب لتحويل ملفات PDF إلى نصوص قابلة للترجمة

## الميزات
- رفع ملفات PDF
- استخراج النصوص
- الترجمة الفورية
- تنزيل ملفات Word
- نسخ النص إلى الحافظة

## كيفية الاستخدام
1. ارفع ملف PDF باستخدام الزر أو السحب والإفلات
2. انتظر حتى يتم استخراج النص
3. اختر خيارات الترجمة إذا لزم الأمر
4. نزّل الملف أو انسخ النص

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript ES6
- Responsive Design
